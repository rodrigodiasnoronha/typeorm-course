## Description

-   API created in the TypeORM course. This API has all relationship methods, how to configure TypeORM migrations, its Entities and its Repositories.

## Instalation

-   Run:

    ```shell script
    npm install
    ```

-   Run API:

    ```shell script
    npm run dev:serve
    ```

-   API will run on port [localhost:3333](http://localhost:3333)
